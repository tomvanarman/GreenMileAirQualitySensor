# Smart Cities Group Project - 2026

## Project Structure

The structure of the project is as follows, you can adjust this to your liking.

- `docs/`: This is the public documentation for the project.
- `teamdocs/`: This is the team documentation for the project, this will not be publically
  accessible.
- `docker/`: This is the docker configuration for the project.
- `embedded/`: This is the embedded software for the project.
- `README.md`: This is the README file for the project.
- `.env`: This is the environment variables for the project.
